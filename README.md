# render-js
The open-source graphical library
## Getting started
1. Make a new html file
```
<html>
  <head>
  </head>
  <body>
    <canvas id="canvas" width="640" height="480"></canvas>
    <script src="https://raw.githubusercontent.com/sapbotgit/render-js/main/render.js"></script>
  </body>
</html>
```
